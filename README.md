# Тестовые задания на стажировку QA

В данном репозитории представлены выполненные тестовые задания для отбора на стажировку QA.

Содержимое:
- Тест-кейсы для сценария оплаты банковской картой через платежный шлюз
- Автотест Playwright (проверка заголовка playwright.dev) с запуском минимум в 2 браузерах
- Решение задачи по теории вероятности

## 1) Тест-кейсы
Файл: `test-cases.md`

## 2) Playwright
Папка: `playwright/`

### Запуск
```bash
cd playwright
npm install
npx playwright install
npm test
```

## 3) Теория вероятности
Файл: `probability.md`
